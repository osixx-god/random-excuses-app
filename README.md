🎭 Random Excuse Generator
A sleek, responsive React web application that generates creative excuses for any situation. Built with Vite, React, and deployed via Netlify.

🚀 Live Demo
View the Live App Here: https://elegant-shortbread-1f6c57.netlify.app/

✨ Features
Instant Excuses: Generates random excuses with a single click.

Category Filtering: Choose excuses for office, party, or family situations.

Responsive Design: Optimized for both mobile and desktop viewing.

🛠️ Tech Stack
Frontend: React.js

Build Tool: Vite

Deployment: Netlify
